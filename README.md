A project to practice presenting data with the Highcharts charting library, exploring and visualizing the demographics of characters in Barbie movies. View the visualizations [here](https://charlottehoyt.github.io/Barbie-Data-Visualization/)!

<br>

**References:**
* Barbie dataset from [Ana Cartola on Kaggle](https://www.kaggle.com/datasets/anacartola/the-barbie-cinematographic-universe-barbieverse/data).
* Barbie font generated with [TextStudio](https://www.textstudio.com/). 
* Barbie color palette from [SchemeColor](https://www.schemecolor.com/barbie-doll-pinks.php).
